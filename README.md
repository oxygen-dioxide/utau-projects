# utau-projects

#### 介绍
oxygen-dioxide歌声合成工程配布，提供ust、nn、sk、dv、带歌词mid、lrc文件

#### 使用规约
允许二次配布，或转换为其他乐谱格式二次配布，但必须提供本页面链接（https://gitee.com/oxygendioxide/utau-projects）

使用这里的工程制作的作品允许投稿至各大平台，但必须注明工程来源：oxygen dioxide
